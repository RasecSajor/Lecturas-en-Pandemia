# Lecturas-en-Pandemia

Analizar la base de datos de uno de los servicios para amantes de los libros con la finalidad de identificar patrones de publicación, contenido y comportamiento de los usuarios. Esta información será utilizada para generar una propuesta de valor para el desarrollo de un nuevo producto, aprovechando el aumento en el consumo de lectura debido al confinamiento ocasionado por el coronavirus.

El análisis incluye:
1. Determinar contenido moderno publicado después del 01 de enero de 2000.
2. Medir las métricas de calificación promedio y número de reseñas para el catálogo de libros.
3. Identificar a la editorial con mayor número de publicaciones de libros.
4. Identificar al autor más popular del catálogo de libros.
5. Determinar el promedio de reseñas que dejan los usuarios.

Tecnologías utilizadas:
- SQL: joins y subqueries para extracción y transformación de datos desde una base de datos relacional.

Consideraciones: 
El archivo "CA.pem" contiene el archivo .ipynb para trabajar con la base de datos.
